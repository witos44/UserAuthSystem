🚀 User Auth System – Fullstack Monorepo

A modern authentication starter built with:

- React + Vite + TailwindCSS + shadcn/ui
- Drizzle ORM
- Node.js backend (monorepo)

## Requirements

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install
cp .env.example .env
npm run dev
```

App runs on http://localhost:5173

## Production Build

```bash
npm run build
npm run preview
```
